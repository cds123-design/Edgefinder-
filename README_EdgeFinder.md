
# EdgeFinder (Streamlit)

This Streamlit app compares our prediction model to DraftKings odds.

## Steps

1. Get an API key from https://the-odds-api.com/
2. Install requirements:

```bash
pip install streamlit requests python-dateutil pandas numpy
```

3. Run locally:

```bash
export ODDS_API_KEY=your_key_here
streamlit run edgefinder_streamlit.py
```

Or upload both files to GitHub and deploy free at https://share.streamlit.io
